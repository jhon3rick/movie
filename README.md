## Available Scripts
yarn para instalar paquetes y yarn start para correr proyecto

## Construcción
- Proyecto creado con el stack react-router-dom, redux, redux-thunk, reselect, immutableJS, sass entre otros

- Almacenamiento de Token por sessionStorage y favoritos por localStorage

- Provider de peticiones al servidor (apiService) centralizando las rutas de peticiones REST en el archivo ./src/apiRoutes

- Iconos vectorizados en SVG y convertidos en fuentes con la herramienta icomoon

- archivo principal de _theme para los estilos globales

- No se usaron librerías o componentes de terceros para los elementos del DOM

* Pido disculpas el proyecto no esta completo falta mucho diseño de las vistas, validación de los inputs en Login entre otros, me enfoque en las mejores practicas y estructura a nivel de código.
